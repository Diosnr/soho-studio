# Soho Studio

Minimal black and white fashion brand website.

## Live Site
https://diosnr.github.io/soho-studio/

## Pages
- **Home** (`index.html`) — Main landing page with hero, about, and collection sections
- **Login** (`login.html`) — Clean login form with email and password fields

## Design
- Pure black and white color scheme
- Minimal, modern typography
- Responsive grid layout
- Clean forms with no clutter

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages deployment

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required

## Notes
- Login form includes frontend validation
- Backend authentication needs to be integrated
- Collection items use placeholder boxes (ready for product images)
